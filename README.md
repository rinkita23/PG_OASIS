# PG Oasis – PG Finder Web Application

## Project Overview

PG Oasis is a web-based platform developed to help students and working professionals easily find Paying Guest (PG) accommodations. The system provides a centralized place where users can browse different PG options, view detailed information, and explore facilities before making a booking decision.

The platform simplifies the process of searching for PGs by displaying organized listings that include rent details, location, amenities, and images. Instead of visiting multiple places physically, users can compare different accommodations online and choose the most suitable option.

This project demonstrates the implementation of a full-stack web application using PHP, MySQL, HTML, CSS, and JavaScript.

---

## Features

* Browse multiple PG listings
* View PG details including rent, location, facilities, and images
* Simple and user-friendly interface
* Organized display of accommodation information
* Booking or inquiry option for users
* Responsive web design for better user experience

## Installation and Setup

1. Install **XAMPP**
2. Clone the repository

```
git clone https://github.com/rinkita23/PG_OASIS.git
```

3. Move the project folder to

```
xampp/htdocs/
```

4. Start **Apache** and **MySQL** from XAMPP

5. Import the database file in **phpMyAdmin**

6. Open the project in browser

```
http://localhost/pgwebsite
```
